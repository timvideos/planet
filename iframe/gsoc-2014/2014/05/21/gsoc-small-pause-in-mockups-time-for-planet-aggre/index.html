As I mentioned before I have started working on mockups for new website, but then community ask me if I can add feature to existing website to have all news aggregated from developers blog. I have small problems with my internet connection today and I couldn't find any answers in the internet so I had to find solution without any help.

It is easy task if you have newest Jekyll. Simply you should use planet.rb from https://github.com/pote/planet.rb to generate news and everything should work out of the box. But it seems that it is not so easy on old Jekyll like ours (which is 0.12.1). On Jekyllrb.com you have only newest documentation and configuration and commands has changed a lot since 0.12.1 release, so currently I am searching the Internet for documentation, I hope I will find it soon.

Meanwhile I would like to present you first version of new mockup for Tim Videos website:


As I said I would like to create modular website with blocks like this. Number of columns that you can see on the screen will adapt to screen resolution, so you will always see content, with full-width website.
