I would like to apologize, that I haven't written any summarize post during last 2 weeks, but I wanted to end one big part of my project and then summarize it in bigger post. I am doing it right now. :)

Last weeks I spend working with GitHub Issues and Supybot configuration. I did a lot of research and read a lot of articles about organizing work with issues, about supybot configuration, etc.
Then I started working with supybot, after few days I created final configuration for RSS changes notification with Supybot and added it to my GitHub: https://github.com/pinoss/supybot-configuration

After working with Supybot I started working with GitHub Issues. I found nice applications and articles about Issues and I wrote about them here: http://pinoit.blogspot.com/2014/07/gsoc-snippet-04072014.html

Then I started working with Issues, I created some mockups and started creating filter/search engine.

And I created GitHub Issues plugin for Jekyll to download all issues and generated and presented them on Issues page, after that I added input autocomplete for author, assignee and title to make search easier.

Then I started refactoring Javascripts and I changes all variable names in JavaScripts to camelCase, then I was ready to add first filter engine for author, assignee and title fields.

After that I changed _config.yml configuration and grouped rss settings into single node. After that I added new field to config.yml with list of special filters based on labels of Issues and modified JavaScript filter engine to support these special filters.

Then I added few small filters: labels, state, milestones and changed markdown engine in issues to GitHub Flavored Markdown.


Now I will work on Getting Started deployment process, I will try moving generation of website to Travis CI and then integrate that site with Planet. Then I want to fix some small CSS/JS problems and start working with Integration Planet with social media pages (notifying about new posts). I also have some tasks in backlog and I will work with them. :)
