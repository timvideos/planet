--- title: "[GSoC] What hapenned during this week (week 3) " kind: article created_at: 2014-06-09 19:51:00 UTC author: Maciej (Developer Website) image: https://avatars1.githubusercontent.com/u/2752578?s=50 categories: GSoC-2014 tags: "news developers-page ruby jekyll" layout: post ---This week finally I've shown my work results on getting started page and deployed updated version of it on my website (http://pinoss.github.io), it still need work, but I have a nice page where I can start updating and improving design and content.

I've added also new page to TimVideos Developers Website, which is NEWS. News are displayed via iframe from http://planet.timvideos.us/iframe, design is the same and common user won't see any differences between News and other parts of website.

TimVideos Planet websites has been also updated. I've added fancybox for every image that was added to post and loading posts content asynchronously. Furthermore I've added RSS generators for tags and categories and ability to download OPML and RSS feeds with icon.

TimVideos Developers page has some new features: I've added correct configuration for prose.io, which can be helpful if someone want to edit content on website. User can also download RSS feed from planet by clicking icon on sidebar.

This week I will work on new deploy scripts which will inform developers when generation process failed, add acceptance tests, add ability to publish news to Facebook, Google+ and Twitter, and add liquid tag to generate new panel (box with content) without having knowledge of HTML and bootstrap (user will only define width and markdown file to include and jekyll will generate views).

Thanks for 3rd week!
