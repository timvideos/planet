That was great week, I've learnt a lot and did some code and prepared small website that is already online: http://planet.timvideos.us, but before I did that I spend some time with creating mockups and design for timvideos.us websites. I think I have created something that looks good, is unique and fit into TimVideos community needs.

Currently I spend my time working on design for current content, that is not easy, because I need to think how to fit new design into old content and I also need to think about the future: New design should work good with new content too, I hope that I will prepare something that everyone will like.

During this week I've also updated Jekyll from 0.21 to 2.0 and every plugins that was used on current solution I had to rewrite. I've added prose.io as a content editor for website, but I also noticed that GitHub offers now great content editor out-of-the-box and many people uses GitHub editor instead of prose.io.
Working with prose.io wasn't easy, I've had to deploy small application on Heroku (called Gatekeeper) to authenticate to GitHub account, I've had some bugs like redirection errors, or routing errors, but I resolved them and prose.io works almost correctly (I am saying almost, because of problems with configuration, local prose.io doesn't want to load prose.yml file from repository to load all settings for editor.

I've written also a lot of small plugins for Jekyll and that was the biggest and greatest fun, Jekyll is really easy to extend and writing plugins is very simple and can be done in many ways. That makes every developer happy. :)

I've prepared planet.timvideos.us page, I've added there my custom theme, added categories, and tags. I've added also GitHub activities, avatars fetched from GitHub accounts. I've written an automatic generate script, which runs every minute and updates the planet.timvideos.us. I've prepared 2 generators: OPML generator and iFrame generator, to provide feed in opml and site without sidebar (only news).

Also I've started to using Producteev to manage project and make sure that everything is realized according to plan.

I've learnt a lot from Tim, my mentor Leon and whole community, thank you guys. :)
